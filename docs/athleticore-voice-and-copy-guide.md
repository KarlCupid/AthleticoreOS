# Athleticore Voice and Copy Guide

Athleticore should sound like a capable coach in the athlete's corner. The voice is calm, confident, supportive, serious, human, clear, and direct.

The copy should never feel cheesy, cold, overly medical, fear-based, robotic, or fake motivational.

This guide applies to Today, Train, Plan, Fuel, readiness, body mass, weight class, fight opportunities, onboarding, and recovery flows.

## Voice Principles

### Calm

Use steady language, especially around fatigue, missed sessions, body mass, and risk.

Prefer:

- "Keep today controlled."
- "Let's protect recovery first."
- "This needs more context before Athleticore pushes the plan."

Avoid:

- "Warning! Critical failure."
- "You are overtrained."
- "Your plan is invalid."

### Confident

Give a clear recommendation. Do not bury the athlete in caveats when the system has enough context.

Prefer:

- "Start with the main session and skip extra conditioning today."
- "Fuel before training. Carbs matter more today."

Avoid:

- "It may be advisable to consider possibly reducing load."

### Supportive

Help the athlete understand the adjustment without shame.

Prefer:

- "You missed a session, so the week is being reshaped around what still matters."
- "The trend is unclear, so Athleticore is asking for one more check-in before pushing volume."

Avoid:

- "Compliance is poor."
- "User failed to complete scheduled work."

### Serious

Athleticore is not a hype app. It should respect boxing, fatigue, body mass, recovery, and risk.

Prefer:

- "Show up fresher for sparring."
- "Protect the shoulder today."
- "Do the work that moves the week forward."

Avoid:

- "Crush it!"
- "No excuses."
- "Beast mode."

### Human

Use natural coaching language. Explain cause and effect plainly.

Prefer:

- "You're carrying a bit more fatigue today, so we're trimming the extra work and keeping the main session sharp."

Avoid:

- "Training load has been adjusted based on readiness inputs."

## Copy Rules

### Lead With The Decision

Say what the athlete should do before explaining the model.

Good:

"Keep the main session sharp. You're carrying fatigue from sparring, so extra lower-body work is trimmed."

Weak:

"Readiness inputs and training stress indicate a moderate load adjustment."

### Explain The Why

Every major recommendation should connect to at least one athlete-facing reason:

- Phase
- Fight date
- Protected workout
- Readiness
- Recent training
- Fueling need
- Body-mass trend
- Safety flag
- Missing data

Good:

"We moved heavy lower-body work away from sparring so you can show up fresher and avoid stacking too much stress on the same day."

Weak:

"Workout conflict detected with protected anchor."

### Translate Metrics Into Coaching

Use metrics as supporting detail, not primary copy.

Good:

"Your recent workload jumped, so today should stay controlled."

Detail:

"Workload ratio: 1.32"

Weak:

"ACWR caution threshold exceeded."

### Treat Missing Data As Unknown

Do not imply the athlete is safe when important data is missing.

Good:

"Athleticore needs today's check-in before pushing intensity."

Weak:

"No readiness issue detected."

### Make Safety Firm But Not Alarmist

Safety copy should be direct and protective.

Good:

"This target looks too aggressive for the time available. Athleticore won't build a risky cut around it. Let's look at safer options."

Weak:

"Weight-class target classified as unsafe."

### Keep Boxing Context Specific

Use boxing-aware language when relevant.

Good:

"Sparring is the anchor today. Strength work stays light so you can show up fresher."

Weak:

"High-intensity activity detected."

## Preferred Vocabulary

Use:

- Mission
- Focus
- Anchor
- Protected workout
- Show up fresher
- Keep it sharp
- Trim extras
- Pull back
- Build gradually
- Fuel today
- Recovery priority
- Safer option
- Needs more context
- What changed
- Why it changed

Avoid as primary copy:

- Compliance
- Protocol
- Optimization
- Glycolytic
- Deficit class
- Intervention state
- Snapshot invalidation
- Classification
- Target violation
- Unsafe cut
- Redline
- Failure

Technical terms may appear in detail views, diagnostics, or developer-facing copy, but they should not carry the main athlete-facing message.

## CTA Rules

CTAs should be short, concrete, and action-oriented.

Good CTAs:

- Start session
- Log check-in
- Log fuel
- Review plan
- View safer options
- Update fight details
- Open weight-class plan
- Start recovery

Avoid:

- Generate recommendations
- Execute adjustment
- Resolve compliance
- Confirm protocol
- Dismiss risk

For safety-related CTAs, avoid language that pressures the athlete to override caution.

Prefer:

- "Review safer options"
- "Adjust today"
- "Log symptoms"

Avoid:

- "Proceed anyway"
- "Ignore warning"

## Surface-Specific Guidance

### Today

Tone: direct, focused, supportive.

Pattern:

"Today: [focus]. [Why]. [Next action]."

Example:

"Today is about showing up fresh for sparring. Strength work stays light, and carbs matter before and after training. Start with your check-in."

### Train

Tone: specific and grounded.

Pattern:

"Do [session]. Keep [effort]. Avoid [risk]."

Example:

"Do the main strength work, but keep top sets controlled. Your readiness is lower, so skip extra finishers."

### Plan

Tone: orienting and strategic.

Pattern:

"This week is shaped around [anchor/phase/fight]. [What changed]."

Example:

"This week is shaped around Thursday sparring. Heavy lower-body work moved earlier so the hard days do not stack."

### Fuel

Tone: practical and performance-centered.

Pattern:

"Fuel for [training demand]. Prioritize [food/hydration/recovery]."

Example:

"Today needs more fuel. You have high-output work planned, so carbs matter more before and after training."

### Body Mass And Weight Class

Tone: protective, firm, non-alarmist.

Pattern:

"This target is [feasible/risky/too aggressive]. Athleticore will [safe action]."

Example:

"This target looks too aggressive for the time available. Athleticore won't build a risky cut around it. Let's look at safer options."

### Fight Opportunities

Tone: boxing-aware and adaptive.

Pattern:

"This fight is [tentative/confirmed/short-notice/rescheduled/canceled]. Athleticore is adapting [training/fuel/body mass/recovery] while preserving your journey."

Example:

"This is a short-notice opportunity, so Athleticore is protecting freshness first and trimming anything that does not help fight readiness."

## Safety Copy Rules

Never recommend:

- Aggressive dehydration
- Severe calorie restriction
- Extreme fluid restriction
- Diuretics, laxatives, vomiting, sweat suits, or sauna-driven cutting
- Ignoring dizziness, fainting, acute pain, illness, injury, REDs-style risk, under-fueling, or poor readiness

When safety conflicts with performance goals, safety wins.

Use:

"Athleticore won't build a risky plan around this."

Do not use:

"Push through."

## Copy Review Checklist

Before shipping athlete-facing copy, check:

- Does it say what to do?
- Does it explain why in plain language?
- Does it connect to the athlete's actual context?
- Does it avoid shame and hype?
- Does it avoid sterile subsystem language?
- Does it treat missing data as unknown?
- Does it avoid unsafe body-mass framing?
- Does it preserve phase continuity?
- Does it sound like a serious coach, not a generic dashboard?
